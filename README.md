# Calculadora Python

Este repositório contém uma calculadora simples desenvolvida em Python, criada como projeto de estudo e também para compor meu portfólio.

## Objetivo

O projeto tem como finalidade demonstrar conceitos básicos de programação em Python, como:

- uso de funções
- organização de arquivos em módulos
- interação com o usuário no terminal
- tratamento de erros simples

## Funcionalidades

A calculadora realiza as operações básicas:

- soma
- subtração
- multiplicação
- divisão

Além disso, ela trata a divisão por zero com uma mensagem de erro amigável.

## Estrutura do projeto

- [CalculadoraPython/calculadora/calculadora.py](CalculadoraPython/calculadora/calculadora.py): interface interativa da calculadora no terminal
- [CalculadoraPython/calculadora/funcoes.py](CalculadoraPython/calculadora/funcoes.py): funções matemáticas utilizadas pela calculadora
- [CalculadoraPython/calculadora/teste.py](CalculadoraPython/calculadora/teste.py): exemplos rápidos de uso das funções

## Como executar

No terminal, execute:

```bash
python CalculadoraPython/calculadora/calculadora.py
```

Se preferir, também é possível testar as funções diretamente com o arquivo de exemplo:

```bash
python CalculadoraPython/calculadora/teste.py
```

## Observação

Este é um projeto simples, pensado para prática e apresentação, com foco em clareza e organização do código.
